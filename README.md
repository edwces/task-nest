# task-nest
todo like app similiar to todoist built with nest.js and next.js

Features:
- create / list / edit todo
- Dashboard
- Tags
- sorting todos
- JWT refresh token Auth
- forgot password / changing password

## Tech Used:

### Client
- Next.js
- react-query
- mantine
- zustand
- framer motion

### Server
- nest.js
- mikro-orm
- redis
- postgresql
- docker
